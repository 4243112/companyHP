## 🎨 デザインコンセプト

- **フッター**: ダークグリーン背景 (#627264) + クリーム色文字
- **テーマ**: プロフェッショナル・ビジネス向け

### 個人向けサイト (kojin/)
- **ヘッダー**: 白背景 + 緑文字

- **テーマ**: 親しみやすい・一般家庭向け

## 🔗 ページ間リンク構造

### 法人 ⇄ 個人 の相互リンク
```
法人サイト (houjin/)
  └─ ヘッダー「個人のお客様」→ ../kojin/TP.html

個人サイト (kojin/)
  └─ ヘッダー「法人のお客様」→ ../houjin/TP.html
```

### 法人サイト内リンク
```
TP.html (トップページ)
  ├─ Overview.html (企業概要)
  ├─ Business.html (事業内容)
  ├─ Achievement.html (実績)
  └─ inquiry.html (お問い合わせ)
        └─ check-inquiry.html (確認ページ)
```

### 個人サイト内リンク
```
TP.html (トップページ)
  ├─ trouble.html (緊急トラブル)
  ├─ service.html (設備交換)
  ├─ inquiry.html (お問い合わせ)
  │     └─ check-inquiry.html (確認ページ)
  └─ ../houjin/Overview.html (企業概要へ)
```

## 📝 主要な技術仕様

### CSS構成
- **all-style.css**: 全ページ共通（ヘッダー、フッター、基本レイアウト）
- **ページ別CSS**: 各ページ固有のスタイル

### JavaScript構成
- **all.js**: スクロールトップボタンの制御
- **ページ別JS**: メニュー展開、フォーム処理など

### 使用ライブラリ
- Font Awesome 6.4.0（アイコン）
- Google Fonts（Material Symbols）

## 🎯 特徴的な機能

1. **メガメニュー**: 個人サイトのドロップダウンメニュー
2. **スクロールトップボタン**: 全ページで実装
3. **レスポンシブデザイン**: スマホ・タブレット対応
4. **ホバーエフェクト**: ナビゲーション要素に統一された効果
5. **相互リンク**: 法人⇄個人サイト間のスムーズな移動
